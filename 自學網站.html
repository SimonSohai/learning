討論與自學網站

Coursera


官方網站： https://www.coursera.org/

我所修的課程： https://www.coursera.org/learn/c-bian-cheng/home/info

Coursera是個全球性的大型自學網站，課程大多由各國知名大專院校所設立，舉凡美國耶魯大學、英國劍橋大學、澳洲墨爾本大學、中國北京大學等
這些課程相對較理論及學術導向，課程也比較多實務操作及經驗分享。

GitHub


官方網站： https://github.com/

Github 是一個透過協同合作方式的軟體開發平台，它提供了問題討論、版本控管、持續整合與持續部屬…等功能，讓團隊成員以撰寫程式或文件進而幫助人們解決問題。
我可以在上面的開源專案給予建議、舉報錯誤、提出問題或提供解決方法，貢獻心力給開源專案，協助有需要的人們解決他們的問題。

Reddit


官方網站： http://www.reddit.com/r/programming

可以和其他程式工程師討論一些編程開發的問題和當前的IT熱點資訊，reddit是一個用戶粘性比較強的開放社區

CodeProject

官方網站： http://www.codeproject.com/

在codeproject里，可以將代碼分享給朋友們。和其他社區不同的是，codeproject更加側重軟體開發，比如C, C#, C++, Java, Lisp等開發。

Hacker News

官方網站： https://news.ycombinator.com/news

可以實時看到編程界中發生的任何事情，包括一些學習的資源和教程。也可以從用戶提交的資料中找到適合自己的學習資源

DZone

官方網站： http://www.dzone.com/links/index.html

DZone是一個允許用戶分享最新IT新聞和編程資源的社區，這是一個文章內容驅動的社區

Bytes

官方網站： http://bytes.com/

Bytes是一個傳統的開發社區，新手和專家都可以在裡面討論一些關於軟體開發、資料庫開發以及網路和系統管理的問題。

DaniWeb

官方網站： http://www.daniweb.com/

DaniWeb以前是專為市場營銷業務分析人員設計的，但是現在主要為各個年齡段的Web開發者和程式工程師服務。

Dream In Code

官方網站： http://www.dreamincode.net/forums/

主要也是以分享的方式讓人觀摩以學習的開放型社區，內容涵蓋 Java, C++, VB.NET等，學習資源相當豐富。

Tech.Pro

官方網站： http://tech.pro/

Tech.pro是一個關注科技領域發展的相關資訊的社區，Tech.pro是一個獲取資源和教程的入門網站，包含一些技術教程、討論等。

Pineapple

官方網站： http://pineapple.io/

但是這裡也包含很多有用的工具、教程以及類庫資源等。

Lobsters

官方網站： https://lobste.rs/

這是一個用戶邀請制的開發社區，是一個真正「面向社交」的社區，你可以在裡面找到大量的關於編程的任何討論。
